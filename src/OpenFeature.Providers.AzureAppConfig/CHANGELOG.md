# Changelog

## [Unreleased]

### Added
- Initial implementation of Azure App Configuration provider for OpenFeature
- Support for Azure App Configuration feature flags
- Support for regular configuration key-value pairs
- Percentage-based feature rollouts
- User and group targeting
- Integration with Azure.Data.AppConfiguration SDK
- Support for all OpenFeature data types (boolean, string, integer, double, structure)
- Configurable prefixes for feature flags and configuration keys
- Label-based filtering support
- Comprehensive error handling and type safety
